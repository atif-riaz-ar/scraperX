### System Setup
- Download dependencies
    ```bash
        pip install -r requrements.txt
    ```
- Create a `.env` file with the following variables:
    ```bash
        LINKEDIN_USERNAME=**************
        LINKEDIN_PASSWORD=**************
    ```
- Run the scraper
    ```bash
        python main.py
    ```
